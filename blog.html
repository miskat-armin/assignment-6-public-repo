Discuss the scope of var, let, and const
var: Function-Scoped: Variables declared with var are function-scoped, which means they can only be accessed within the function in which they are declared. Global-scoped:if used outside of a function they act as global variable, that are accessable throught the entire program. let: Block-Scoped: Variables the declared with let are block-scoped, meaning they can only be used inside in the block. const: Block-Scoped: Like let, variables declared with const are block-scoped. however they act like constant values where's let acts as a mutuable variable.
Tell us the use cases of null and undefined
null: null is a primitive value that represents the absence of a value or the intentional absence of an object or variable.

let someValue = null; // indicating that 'someValue' has no meaningful value explicitly.

undefined: undefined means a variable that has not been assigned a value.
When a function doesn't return a value, it implicitly returns undefined.

function doSomething() {
// No return statement, so 'undefined' is returned.
}

What do you mean by REST API?

A REST API (Representational State Transfer Application Programming Interface) is a way for software applications to communicate over the internet using protocols and conventions. It allows systems or application the ability to request and exchange data or perform actions on remote servers. REST APIs are used for CRUD operations using HTTP methods such as GET, POST, PUT, and DELETE to interact with data.
